# OFFICIUM

> *una macchina da liturgia industriale*

Esperimento web sonoro all'incrocio fra ritmica drum &amp; bass e
sintetica EBM/dark-pop dei primi anni Ottanta. Scritto in HTML, CSS e
JavaScript puri, senza framework, senza dipendenze runtime, senza
build step. Tutta la sintesi viene generata in tempo reale con la
**Web Audio API** nativa.

In costruzione.

## Avviare in locale

```bash
npm start
# → http://localhost:4173
```

In alternativa qualunque server statico funziona; non c'è build da
fare.
